# Especificaciones - Lamen Magic Experience

## Concepto
Un bar de ramen de autoservicio y market de snacks asiáticos exóticos donde el cliente prepara su propia comida en estaciones equipadas.

## Funcionalidades Web
- Guía visual del proceso "Elige, Cocina, Disfruta".
- Catálogo destacado de más de 30 variedades de ramen.
- Sección de snacks y microondas.
- Información de contacto y ubicación en Getafe.

## Diseño
- **Estética:** Cyberpunk/Moderno asiático con gradientes rojos y dorados.
- **Tecnología:** React + TypeScript + Vite.
- **Despliegue:** GitHub Pages vía GitHub Actions.
